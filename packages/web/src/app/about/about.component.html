About

This is Lomodoro, a web app that allows you to use pomodoro technique for productivity while listening to open source lofi music. This was created for a fun personal project so please enjoy it as I have enjoyed making it.

1. Set your work/break times using the settings icon on the pomodoro timer.
2. Click start to begin your timer. Once the timer counts down you can begin the next cycle.
3. Click the next button and hit start to begin the next cycle.
4. You can change the time settings at anytime using the settings icon. You can also switch cycles any time by clicking the next button.
